# Universal AI SaaS Boilerplate — Agent Instructions

**ВНИМАНИЕ АГЕНТУ:** 
Этот проект является **универсальным стартовым шаблоном (Boilerplate)** для новых B2B SaaS проектов.
Текущее название "HexaCash" и все данные в `memory-bank` оставлены **исключительно в качестве примера**. 

Твоя точка входа для начала работы над новым проектом — это файл `QUICKSTART.md` в корневой директории. 
**Немедленно открой и прочитай `QUICKSTART.md`**, чтобы понять, как развернуть проект, инициализировать его под новое название и какие вопросы задать пользователю.

---

## Правила работы с Memory Bank (после инициализации)

- В этом проекте используется **Memory Bank** — система постоянного контекста.
- После значимых изменений в проекте — **обновляй соответствующие файлы Memory Bank**.
- Если что-то противоречит Memory Bank — уточни у пользователя, не игнорируй расхождение.
- `activeContext.md` и `progress.md` — обновляются чаще всего, отражают текущее состояние.
- `projectbrief.md` — источник истины по продукту; не противоречь ему без явного решения пользователя.
