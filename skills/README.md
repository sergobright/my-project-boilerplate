# Development Skills

Эта папка содержит скрипты ("Скиллы" / "Skills"), которые расширяют возможности ИИ-агентов, работающих в вашей IDE (например, Cursor, Windsurf, Claude Code, Gemini).

ИИ-агент может вызывать эти скрипты для автоматизации рутинных задач в проекте, вместо того чтобы вручную подбирать команды.

## Доступные скиллы

* (Здесь можно размещать bash/python скрипты с сайта `skills.sh` или свои собственные).
* Чтобы агент понимал, как их использовать, каждый скрипт должен содержать описание в начале файла или иметь понятное название.

### Пример добавления скилла
Если вы скачали скрипт `find_todos.sh` с `skills.sh`, просто положите его в эту папку. Агент сможет самостоятельно находить его и запускать по вашей просьбе:
`"Найди все TODO в проекте с помощью скилла"` -> Агент запустит `./skills/find_todos.sh`.
