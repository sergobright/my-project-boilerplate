## Принятые решения

| Вопрос | Решение |
|---|---|
| Структура репозитория | **Монорепо** (pnpm workspaces) |
| Бэкенд | **Next.js API Routes** (App Router) — отдельного Node.js-сервиса нет |
| ORM | **Prisma** |
| AI-модель агента | Не фиксируется в коде — **выбирается и конфигурируется в N8N** |
| Получение сообщений из Avito | Через **N8N** (N8N на удалённом сервере, приложение получает события через webhook) |
| N8N | Self-hosted на удалённом сервере (не в docker-compose проекта) |
| Среда разработки | **Docker** — всё внутри контейнера, ничего на хосте |
| Порт приложения | **4000** (3000 и 3100 заняты другими сервисами на хосте) |
| PostgreSQL порт | **5433** на хосте (5432 занят) |
